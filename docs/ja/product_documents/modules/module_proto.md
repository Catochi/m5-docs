# M5Stack PROTO モジュール

## 概要

M5Stack PROTO モジュールは、各M5Stack Coreと接続可能な30ピンソケットとユニバーサル基板が一体になった試作用モジュールです。あなたのアイデアを自由に試作することができます。

## 特徴

- フレキシブルなブランクホール
- M5Stack Coreと互換性のある30ピンソケット
- 温湿度センサー DHT12 付属 (キット購入時)
