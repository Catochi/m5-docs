# M5Stack COMMU モジュール

## 概要

M5Stack COMMU モジュールはRS-485通信(SP3485)、CAN通信(コントローラ: MCP2515, トランシーバ: TJA1051)、I2C通信が可能なモジュールです。

## 特徴

- 1x RS-485ポート
- 1x CANポート
- 2x I2Cポート
- 1x TTLポート
