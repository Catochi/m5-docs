# COLOR SENSOR ユニット

## 概要

COLOR SENSOR ユニットは内蔵されたTCS3472により、対象物表面の色を検出する事ができます。M5StackはI2Cで値を取得可能です。

## 特徴

- 高精度
- LEGO 互換

## アプリケーション

- RGB LED バックライトコントロール
- 色校正
