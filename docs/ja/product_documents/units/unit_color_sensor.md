# COLOR SENSOR ユニット

## 概要

This is a unit can detecte the color of object surface which integrates TCS3472 (a color sensor). The unit comunicates with M5Core with I2C.


## 特徴

- 高精度
- LEGO 互換

## アプリケーション

- RGB LED バックライトコントロール
- 色校正
