# M5Stack USB Downloader

## 概要

1. USB/UART変換チップCP2104によるESP32/ESP8266のファームウェア書き込みをサポートします。
2. ６ピン@2.54mmバスソケットです。

## ピンマップ

|          Pin#         |           Name            |
| :---------------------|:------------------------- |
|           1           |           GND             |
|           2           |           GPIO0           |
|           3           |           EN              |
|           4           |           TXD             |
|           5           |           RXD             |
|           6           |           3.3V            |

?>メモ M5Stack USB Downloaderには、他のアプリケーション用に2つの予約済みピン（RTS、DTR）があります。

## パッケージ内容

- 1x M5Stack USB Downloader
