# GitとArduino IDEのインストール方法 {docsify-ignore-all}



## Windows

## 1. `Git`のインストール
もし既に `Git` がインストールされている場合は、次のステップへ進んでください。そうでなければ [Git](https://git-scm.com/download/win) をダウンロードし、インストールしてください。
