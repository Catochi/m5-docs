## 概要

USBtoUART変換チップCP2104によるESP32/ESP8266のファームウェア書き込みをサポートします。
送信、受信、パワーのインジケータLED付きです。６ピン@2.54mmバスソケットです。

## ピンマップ

| Pin# | Name  |
|:-----|:------|
| 1    | GND   |
| 2    | GPIO0 |
| 3    | EN    |
| 4    | TXD   |
| 5    | RXD   |
| 6    | 3.3V  |

### メモ

M5Stack USB Downloaderには、他のアプリケーション用に2つの予約済みピン（RTS、DTR）があります。
