## 概要

USB-UART変換チップCP2104によるシリアルポートUSB書き込みをサポートします。ファームウェアの書き込みなどで使用できます。

## ピンマップ

2本のピンが予約済みです（RTS, DTR）

| Pin# | Name  |
|:-----|:------|
| 1    | GND   |
| 2    | GPIO0 |
| 3    | EN    |
| 4    | TXD   |
| 5    | RXD   |
| 6    | 3.3V  |
