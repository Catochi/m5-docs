# JOYSTICK ユニット



## 概要

**<mark>JOYSTICK</mark>**ユニットはゲームコントローラーと同じジョイスティックです。X-Y軸オフセットとボタンクリックの入力を取得できます。

## 特徴

- LEGO 互換ホール
- Grove インターフェース

## ドキュメント

- **サンプルコード**
  - [Arduino](https://github.com/m5stack/M5Stack/tree/master/examples/Unit/Joystick)
