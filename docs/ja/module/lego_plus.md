# LEGO+ モジュール

## 概要

<mark>**LEGO+**</mark>モジュールは、ATmega328を内蔵したDCモーターコントロールモジュールです。LEGO+モジュールはI2Cを介してM5Coreと通信します。I2Cアドレスは<mark>**0x56**</mark>です。このモジュールには、高電圧入力ドライブのモーターが必要な場合に接続できるDC電源ジャックも装備されています。
## 特徴

- DCコネクタ許容入力電圧範囲：6-12V

## パッケージ内容

- 1x LEGO+モジュール
- 1x 10cm LEGOケーブル
- 1x DCジャック
