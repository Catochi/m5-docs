# BATTERY モジュール



## 概要

**<mark>BATTERY</mark>**モジュールは850mAhの大容量バッテリを内蔵しています。

これによりM5Stack単独で動作するポータブルデバイスを開発する事が可能です。

## 特徴

- 850mAh 大容量バッテリ

## パッケージ内容

- 1x BATTERY モジュール
