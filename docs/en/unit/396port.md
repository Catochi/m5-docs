## Description

<mark>3.96port</mark> is a unit that aim to transfer GROVE interface to VH3.96-4Pin.

## Feature

-  VH3.96 interface
-  GROVE interface, support [UiFlow](http://flow.m5stack.com) and [Arduino](http://www.arduino.cc)
-  Two Lego installation holes
