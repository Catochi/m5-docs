## Description

A tool built-in USB-to-UART bridge chip(CP2104) can be connected with UART port for automatically burning firmware.

TXD led, RXD led, Power led and 6pin @ 2.54mm bus sockets.

## PARAMETER

|       PinNumber       |          PinName          |
| :-------------------: |:------------------------: |
|           1           |           GND             |
|           2           |           GPIO0           |
|           3           |           EN              |
|           4           |           TXD             |
|           5           |           RXD             |
|           6           |           3.3V            |
