# M5Stack USB Downloader

中文 | [English](en/product_documents/tools/tool_usb_downloader) | [日本語](ja/product_documents/tools/tool_usb_downloader)

## 描述

内置CP2104芯片的串口转USB烧录工具，使用这款工具，能自动烧录固件。

## 参数

|       管脚号       |          管脚名字          |
| :-------------------: |:------------------------: |
|           1           |           GND             |
|           2           |           GPIO0           |
|           3           |           EN              |
|           4           |           TXD             |
|           5           |           RXD             |
|           6           |           3.3V            |

### 其他

有两个引脚作为预留(RTS, DTR)

## 包含

- 1x M5Stack USB Downloader
