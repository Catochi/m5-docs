## 描述

内置CP2104芯片的串口转USB烧录工具，使用这款工具，能自动烧录固件。

## 参数

|       管脚号       |          管脚名字          |
| :-------------------: |:------------------------: |
|           1           |           GND             |
|           2           |           GPIO0           |
|           3           |           EN              |
|           4           |           TXD             |
|           5           |           RXD             |
|           6           |           3.3V            |
