# 描述

BATTERY模块是一款850mAh的高容量电池模块。当M5Core堆叠了这个电池模块之后，可以做成移动的设备，不再需要USB线一直供电。

# 特性

-  850mAh容量

# 包括

-  1x BATTERY模块
