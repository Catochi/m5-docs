# BTC底座

中文 | [English](/en/product_documents/modules/module_btc) | [日本語](ja/product_documents/modules/module_btc)

# 描述

BTC模块是一款包含DHT12(温湿度)传感器的底座。有了BTC底座之后，可以方便给M5Core充电。

#  特性

-  内置DHT12

#  包含

-  Type-C USB线
-  M3 x 16

#  管脚映射

| DHT12        | ESP32      |
| :----------:  |:------------:|
| SCL          | G22 |
| GND          | GND |
| SDA          | G21 |
| 3V3          | 3V3 |
