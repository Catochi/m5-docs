# M5Stick 上手指南(Arduino)

## 1. 安装`U8g2`库

如果你已经安装了`U8g2`库的话，直接跳到第二步往下执行。
如果还没的话，打开Arduino IDE，并点击`Sketch`->`Include Library`->`Manage Libraries...`，搜索`U8g2`进行安装。
