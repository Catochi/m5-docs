# Accessory LEGO {docsify-ignore-all}

## 描述

**LEGO Cable** 是一款 LEGO 电机连接线.通过该连接线能够兼容M5体系中的LEGO+模块以及BALA平衡车.提供了不同的长度选择，以满足不同的应用需求.

## 特性

-  3种规格: 10cm, 20cm, 30cm
