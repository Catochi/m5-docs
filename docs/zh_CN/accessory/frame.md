# Accessory Frame {docsify-ignore-all}

## 描述

**Frame** , 是一款M5Core固定框架.能够帮助你将设备牢固的固定在墙壁或是其他表面,并为设备提供一定强度的保护外壳.

## 套件清单

-  1x FRAME
-  4x 六角螺钉 KA2x4
-  1x 内六角扳手
